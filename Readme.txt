== Installation ==

1. Upload the entire rss-feed-scroller-widget folder to the /wp-content/plugins/ directory.
1. Activate the plugin through the 'Plugins' menu in WordPress.

You will find the configuration menu in the widgets area.
You find a quick but detailed documentation on http://www.vonrueden-design.de/blog/neuigkeiten/rss-feed-sidebar-scroller-widget.html
